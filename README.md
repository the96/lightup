# 概要
このリポジトリは、らいとあっぷボタンのソースコードです。
らいとあっぷボタンはgithub.ioとCloudflare R2を使って運営されており、ある程度の規模までは無料で運営することができます。

# 構築手順

- このリポジトリをフォーク
- cloudflare R2のPublic Development URLを取得
- index.html の免責事項を編集
- main.jsの `FILE_PREFIX` を置換
- members.csvにメンバー情報を書き込み
  - メンバー情報は `キー, 表示名, X(Twitter)アカウント, YouTubeアカウント` で構成されます
- _メンバー情報のキー_.csvにボイス情報を書き込み
  - ボイス情報は `カテゴリ番号_カテゴリ名, ファイル名` で構成されます
  - カテゴリ番号は表示されず、表示する際の並び順に影響します
  - ファイル名は.mp3を取り除いてボタンの表示名に使われます
- コミットしてmainブランチへpushすると `アカウント名.github.io/lightup` にデプロイされます
